Lattitude Vs Clouds

For this visualization we use matplotlib to quickly create a scatter plot based on the data given. Where each point is plotted first by the Lattitude and then by the density of the clouds at the time the data was pulled from the API.

Here the data runs the gamut for each latitude. One of the interesting things is that it seems as if there is a tendency to report the cloud cover with certian percentages more often. 0%, 20%, 40%, 75% and 100% seem to be the most common ranges used to describe how cloudy an area is.
